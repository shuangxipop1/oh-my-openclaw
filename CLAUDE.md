# oh-my-openclaw

Multi-agent orchestration for OpenClaw AI.

## Skills

- `/autopilot` - Full autonomous execution
- `/ralph` - Persistence loop until completion
- `/ultrawork` - Parallel execution engine
- `/team` - Multi-agent coordination
- `/plan` - Strategic planning
- `/ultraqa` - QA cycling workflow
- `/cancel` - Cancel active modes

## Architecture

- `skills/` - Skill definitions (SKILL.md + __init__.py)
- `bridge/` - CLI bridge for OpenClaw integration
- `agents/` - Agent definitions for multi-agent workflows
