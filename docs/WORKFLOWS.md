# Workflows

## Autopilot Mode

End-to-end autonomous pipeline from idea to code.

1. Architect analyzes requirements
2. Executor implements code
3. Critic reviews implementation
4. Repeat until complete

## Ralph Mode

Persistence loop with verification.

1. Execute current task
2. Verify result
3. If incomplete, retry with learned corrections
4. Repeat until verification passes

## Ultrawork Mode

High-throughput parallel execution.

1. Break task into independent subtasks
2. Execute subtasks in parallel
3. Aggregate results
4. Report completion

## Team Mode

Coordinated multi-agent workflow.

1. Planner breaks down project into tasks
2. Architect designs system
3. Executors implement in parallel
4. Critics review continuously

## Ultraqa Mode

QA cycling workflow for continuous quality assurance.

1. Generate test cases
2. Execute tests
3. Critic analyzes failures
4. Debugger fixes issues
5. Repeat until quality passes
