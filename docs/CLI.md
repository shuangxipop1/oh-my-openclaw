# CLI Reference

## Commands

### `omo setup`

Install or refresh OMO installation.

```bash
omo setup
```

### `omo doctor`

Verify OMO installation.

```bash
omo doctor
```

Checks:
- Node.js version
- OpenClaw installation
- OMO skills directory
- Individual skill files

### `omo help`

Show help information.

```bash
omo help
```
